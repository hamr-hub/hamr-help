# HamR 帮助中心 (help.hamr.store)

HamR 平台的用户帮助中心和文档站。

## 项目信息

- **项目名称**: HamR 帮助中心
- **项目编号**: PROJ-004
- **域名**: help.hamr.store
- **技术栈**: React + TypeScript + Vite + Docusaurus
- **部署**: Cloudflare Pages

## 功能特性

- 产品使用教程
- 常见问题（FAQ）
- 故障排查指南
- 搜索功能（Algolia）
- 用户反馈表单

## 开发

```bash
npm install
npm run dev
```
